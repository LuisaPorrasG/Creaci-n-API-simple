Consumir API REST

Instrucciones para consumir la API REST:

Obtener todos los estudiantes:

Método Ruta
GET /api/students

Obtener un estudiante por ID:

Método Ruta
GET /api/students/:id

Agregar un nuevo estudiante:

Método Ruta
POST /api/students

Eliminar un estudiante por ID:

Método Ruta
DELETE /api/students/:id
